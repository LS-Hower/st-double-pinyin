# 键位

```
Q     W           R     T     Y     SH    CH          P
ang   ao    e     iang  üe    ai    u     i     o     uan
                  uang  uai                     uo    üan

         S     D     F     G     H     J     K     L
   a     ie    en    iao   in    ing   an    iu    ia
                     er                            ua

      Z     X     C     ZH    B     N     M
      ian   eng   ei    ü     un    ou    ong
                        ui    ün          iong
```

将 `O` 键作为零声母键，也就是说：

```
阿(a)   oa
爱(ai)  oy
安(an)  oj
昂(ang) oq
袄(ao)  ow
饿(e)   oe
欸(ei)  oc
恩(en)  od
鞥(eng) ox
耳(er)  of
藕(ou)  on
```

`er` 原本是放在 `C` 键的（`"0.02"` 版），现在改成放在 `F` 键了（`"0.03"` 版）。

那位唯一的用户使用的是 `"0.02"` 版。
